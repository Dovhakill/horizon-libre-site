Turbulent devient Cloud Imperium Games Montréal: un studio web se tourne vers le jeu vidéo

Le studio montréalais Turbulent, spécialisé dans les solutions web, change de nom et d'orientation pour devenir Cloud Imperium Games Montréal, se concentrant sur le développement de jeux vidéo comme Star Citizen et Squadron 42. Cette transition vise à attirer de nouveaux talents et à soutenir l'ambitieux projet Star Citizen.

22 August 2025

Le studio montréalais Turbulent, connu pour ses solutions web, amorce un tournant majeur en devenant Cloud Imperium Games Montréal. Acquis il y a deux ans par Cloud Imperium Group, le changement de nom reflète une réorientation stratégique vers le développement de jeux vidéo, une activité déjà en cours depuis l'acquisition.

Cette transformation vise principalement à attirer des talents dans le secteur du jeu vidéo. Benoît Beauséjour, cofondateur de Turbulent et désormais directeur des technologies pour Cloud Imperium Group, explique que le nom Turbulent était davantage associé à leurs solutions technologiques qu'aux jeux vidéo. Le studio vise à atteindre 150 employés et à devenir un acteur majeur de l'industrie du jeu vidéo montréalaise.

Cloud Imperium Games Montréal travaillera principalement sur le jeu Star Citizen, un jeu de simulation spatiale massivement multijoueur, dont la version finale est prévue pour 2027 ou 2028. Le studio prévoit également le lancement de Squadron 42, un jeu solo narratif se déroulant dans le même univers que Star Citizen, en 2026. Le succès de Star Citizen, avec plus d'un million de joueurs actifs mensuels, devrait faciliter le lancement de Squadron 42. Chris Roberts, PDG de Cloud Imperium Group, souligne le financement unique de Star Citizen par les joueurs, lui permettant de développer le jeu sans les contraintes habituelles des studios de jeux vidéo.

Les cofondateurs de Turbulent, Benoît Beauséjour et Marc Beaudet, ainsi que l'associée Claire Buffet, occupent de nouveaux rôles au sein de Cloud Imperium Group. Marc Beaudet dirigera les studios de Montréal, Manchester, Austin et Francfort, tandis que Claire Buffet sera responsable des opérations et des ressources humaines pour les bureaux nord-américains. Malgré les turbulences chez Air Canada, Chris Roberts n'a pu se rendre à Montréal pour l'inauguration officielle du changement de nom.

Points clés

  • Turbulent devient Cloud Imperium Games Montréal pour se concentrer sur le jeu vidéo.
  • Le studio vise 150 employés et travaille sur Star Citizen et Squadron 42.
  • Le financement unique de Star Citizen par les joueurs permet une grande liberté créative.
